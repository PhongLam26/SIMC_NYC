This supplementary note documents the deterministic mapping from original NYC 311 complaint types to the nine analysis categories used in the main manuscript. The full 320-row mapping is provided in \texttt{tables/complaint\_type\_category\_mapping\_appendix.csv}. The full composition of the residual \texttt{other} category is provided in \texttt{tables/other\_category\_composition.csv}. These files include request counts and request-share fields computed from the NTA-matched 2015--2025 311 request corpus.
